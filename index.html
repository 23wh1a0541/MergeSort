How Merge Sort Works
  1. Divide the unsorted array into n subarrays, each containing 1 element
  2. Repeatedly merge subarrays to produce new sorted subarrays
  3. Continue until there is only 1 sorted array remaining
  4. Merge operation compares elements from two sorted arrays
  5. Take the smaller element and place it in the result array
  6. Continue until all elements are merged into one sorted array
